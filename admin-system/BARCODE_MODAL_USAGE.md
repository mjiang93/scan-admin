# 条码弹窗功能使用说明

## 功能概述

本功能为Print页面的"本体"按钮添加了一个弹窗，用于显示二维码和条形码。弹窗包含以下内容：

1. **二维码**：包含产品信息（PN、Rev、Model、SN）
2. **条形码**：两个条形码，格式为 `S{snCode}IP{projectCode}PA{code09}-001/002`
3. **产品信息**：显示PN、Rev、Model、SN等详细信息
4. **操作按钮**：打印和下载功能

## 技术实现

### 使用的第三方库

- **jsbarcode**: 用于生成条形码
- **qrcode.react**: 用于生成二维码

### 组件结构

```
src/components/BarcodeModal/
├── index.tsx          # 主组件文件
└── index.css          # 样式文件
```

### 主要功能

1. **二维码生成**：
   - 内容格式：`PN:{projectCode};Rev:{techVersion};Model:{factoryCode};SN:{snCode}`
   - 使用QRCodeSVG组件生成

2. **条形码生成**：
   - 使用JsBarcode库生成CODE128格式条形码
   - 两个条形码分别以-001和-002结尾

3. **打印功能**：
   - 打开新窗口进行打印
   - 包含完整的产品信息和条码

4. **下载功能**：
   - 目前为占位功能，可扩展为图片下载

## 使用方法

1. 在Print页面的表格中，点击任意记录的"本体"按钮
2. 弹窗将显示该记录对应的二维码和条形码
3. 可以使用"打印"按钮打印条码标签
4. 点击"关闭"或弹窗外部区域关闭弹窗

## 样式特点

- 响应式设计，支持移动端显示
- 清晰的视觉层次，二维码和条形码分区域显示
- 打印友好的样式，确保打印效果清晰

## 扩展建议

1. **下载功能完善**：可以将条码转换为图片格式下载
2. **模板定制**：支持不同的条码模板和尺寸
3. **批量操作**：支持批量生成和打印条码
4. **条码验证**：添加条码格式验证功能

## 注意事项

- 确保项目已安装 `jsbarcode` 和 `qrcode.react` 依赖
- 条形码生成依赖于记录的完整数据，确保数据字段完整
- 打印功能需要浏览器支持window.open和打印API
