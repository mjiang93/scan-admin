# 条码打印管理系统

## 项目概述

这是一个基于 React + TypeScript + Ant Design 的条码打印管理系统，主要用于管理和打印各种条码标签。

## 主要功能

### 1. 条码打印页面（首页）
- **搜索功能**：支持按产品编码、产品名称、批次号、规格、生产日期、打印状态等条件搜索
- **数据展示**：以表格形式展示条码打印记录，包含产品信息、批次信息、打印状态等
- **批量操作**：支持批量选择记录进行打印或导出
- **单条操作**：支持单条记录的预览和打印
- **状态管理**：显示打印状态（待打印、已打印、已完成）和打印次数

### 2. 用户管理页面
- **用户列表**：展示系统用户信息，包括用户名、昵称、邮箱、角色、状态等
- **用户搜索**：支持按关键词、角色、状态搜索用户
- **用户管理**：支持新增、编辑、删除用户
- **角色管理**：支持管理员和普通用户两种角色

### 3. 系统布局
- **响应式设计**：支持桌面端和移动端适配
- **侧边栏导航**：可折叠的侧边栏，包含条码打印和用户管理两个主要功能
- **顶部导航**：显示系统标题、用户信息和操作菜单
- **主题风格**：采用现代化的蓝色主题设计

## 技术栈

- **前端框架**：React 19.2.0
- **类型系统**：TypeScript
- **UI 组件库**：Ant Design 5.29.1
- **状态管理**：Zustand 5.0.8
- **路由管理**：React Router DOM 6.30.2
- **HTTP 客户端**：Axios 1.13.2
- **构建工具**：Vite 7.2.4
- **代码规范**：ESLint + Prettier

## 启动项目

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 访问地址：http://localhost:5174

## 页面功能说明

### 条码打印页面
- 登录后默认进入此页面
- 提供完整的条码打印记录管理功能
- 支持多条件搜索和批量操作
- 包含打印预览和状态跟踪

### 用户管理页面
- 系统用户的增删改查功能
- 支持角色分配和状态管理
- 提供用户搜索和筛选功能

## 特色功能

1. **无权限校验**：按照要求，暂时不配置页面访问权限校验
2. **响应式设计**：适配不同屏幕尺寸
3. **现代化UI**：采用 Ant Design 5.x 最新设计规范
4. **类型安全**：全面使用 TypeScript 确保代码质量
5. **模块化架构**：清晰的项目结构和组件划分

## 开发说明

- 项目采用函数式组件和 Hooks
- 使用 Zustand 进行轻量级状态管理
- 路由采用动态配置，便于扩展
- 组件设计遵循单一职责原则
- 代码风格统一，使用 ESLint 和 Prettier
